Correlated Themes Across Interviews

🌐 Situational Awareness and Information Sharing

During coordinated attacks, no one has a shared operational overview. Information sharing happens over email and phone, too slowly to coordinate response.

Faster automation in production and grid operations increases exposure, not only to attacks, but also to unintended failures between internal systems.

🔗 Supply-Chain Attacks

🧱 IT/OT Segmentation

Separating business IT from operational OT systems is critical. Isolation during incidents can halt power trading, creating an operational dilemma.

HN
TH

🔍 Real-Time Vulnerability Visibility

Zero-day vulnerabilities are discovered too late. Continuous, automated visibility into current software and firmware weaknesses is missing.

TH
EG

🛡️ Baseline Security Is Undervalued

The sector chases new technology and AI, while practical fundamentals such as patching, updates, and access control remain the strongest protection.

EG
HN

🚨 Alert Fatigue and False Positives

High alert volume makes critical detection harder. Operator competence is as important as tooling quality.

EG
TB

🏗️ Centralization as a Vulnerability

A unified control system is efficient but becomes an ideal target. If trust in telemetry is lost, operations fall back to manual control.

Key Statements

"Baseline security is probably what protects you best - make sure it is in place. That may be the single most important thing you do."
Esben Gudbrandsen - CISO, Statnett
"Sabotage is the biggest challenge. It is very easy to enter critical systems because some basic security controls are not in place."
Haugstoyl - Director, NSM
"I would ask for a filter that prevents wrong setpoints from being sent from our aggregates. With such a filter, I would feel even more confident about increased automation."
Torbjorn - Operations Control Center Manager, Hafslund Kraft
"Severe zero-day vulnerabilities are discovered too late. I want a super-software that continuously scans and gives near real-time vulnerability status."
Tor Heiberg - Security Manager
"If we can no longer trust the data we see, we can no longer trust the control system. Then we shut everything down and run stations manually by phone."
Vetle - Grid Control Center, Glitrenett
"We handle symptoms instead of understanding what is actually happening. We lack an environment that can connect the dots and understand the real situation."
Erland Kolstad - CISO, Aenergi
